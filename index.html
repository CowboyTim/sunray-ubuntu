sunray-ubuntu by CowboyTim

SunRay software repackager for debian

This project is ment to make SunRay software installs maintainable on debian systems. The SunRay server software from Oracle is installable on linux and only supported on SUSE/RedHat. It repackages the SunRay software into a deb file that can be installed with dpkg or even apt-get when put in a repository.

Source

The source is on github and can be found here

Dependencies

SunRay Server Software 4.2 (srss 4.2) from Oracle (formerly SUN). You also need at least a debian squeeze install amd64. The newest unstable is also supported with the newer linux-image 3.2.0-2-amd64 version.

Build srss debiab dpkg package

on debian unstable, as a regular user: 

    $ sudo apt-get install fakeroot alien build-essential linux-headers-3.2.0-2-amd64
    $ bash ./srss_install <path to srss unzip> [<4.1|4.2>]

Install built srss dpkg

The build script makes a correct Packages.gz and deb file in ~/srss. One can add this as an apt repository or copy this over to a webserver and add that as a repository: 
    $ sudo echo "deb file:///home/test /" > /etc/apt/sources.list.d/srss.list
Or: 
    $ sudo echo "deb http://some.repository.org/debs/srss /" > /etc/apt/sources.list.d/srss.list
You also need extra packages, so make sure you've added all for ubstable: 
    $ sudo echo "
    deb http://ftp.debian.org/debian/ unstable main non-free contrib
    deb-src http://ftp.debian.org/debian/ unstable main non-free contrib
    " > /etc/apt/sources.list.d/unstable.list
And update/install: 
    $ sudo su -
    # apt-get update
    # apt-get install srss
    # update-rc.d zsunray-init enable  3 5
